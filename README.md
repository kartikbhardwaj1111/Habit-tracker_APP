## �️ **Quick Start**

<div align="center">

### **⚡ Get Running in 60 Seconds**

</div>

```bash
# 🚀 Clone & Install
git clone https://github.com/kartikbhardwaj/ai-habit-tracker.git
cd ai-habit-tracker && npm install

# 📱 Run on Your Device
npx expo start
# Scan QR with Expo Go app

# 🖥️ Or Run in Browser  
npx expo start --web
```
